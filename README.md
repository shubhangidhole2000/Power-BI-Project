# Power-BI-Project
Power BI is used to visualize and analyze hospital performance data through interactive dashboards for better decision-making.

## Hospital Performance Analysis Dashboard (Power BI Project)

## Overview
This project visualizes hospital operations and patient data to improve healthcare management and performance monitoring.It provides key insights into appointments, treatment trends, payment methods, and patient demographics using an interactive Power BI dashboard.

## Tools Used
- Microsoft Power BI
- Excel (for data source)
- Power Query (for data cleaning)
- DAX (for calculated measures)

## Project Workflow
1.	**Data Collection:** Imported healthcare dataset from Excel.
2.	**Data Cleaning:** Removed nulls, standardized date formats, and corrected department names.
3.	**Data Modeling:** Created relationships between patient, treatment, and billing tables.
4.	**DAX Calculations:** Added KPIs like Total Appointments, Revenue, and Missed Visit Rate.
5.	**Dashboard Design:** Built interactive visuals for management insights.

## Key Insights
- **Central Hospital** has the highest number of patients and revenue.
- **April** recorded the most treatments.
- Chemotherapy and X-Ray are the most used treatments.
- **Credit Card** is the most common payment method.
- Pediatrics Department has the highest treatment cost.
